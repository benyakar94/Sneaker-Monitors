General Parameters

There are a few standard parameters that need to be set for all monitors. Here, I will mention how they should be set up and ways to customise them.

Discord Webhook:
This should be pasted from your Discord Channel. It should be formatted as such:

WEBHOOK = "https://discord.com/api/webhooks/..."

Discord Bot - Username:
This is the username that the bot user will use.

USERNAME = "Supreme Monitor"

Discord Bot - Avatar:
This is the avatar that that bot user will use on Discord. You can provide either a url to the image or the path to the image (if the image is stored locally).

AVATAR_URL = "path/to/image.jpg"

Discord Bot - Colour:
This is the colour that that bot will use on Discord. The code must be a decimal number. You can use a colour picker such as ColorHexa to choose a colour, but then convert the hexadecimal number to decimal.

COLOUR = 12845619

Delay:
This is the delay in seconds between requests. Sometimes higher/lower delays may work better.

DELAY = "1"

Keywords:
This is an optional parameter which will allow you to only get notified of specific products that contain any of these keywords within its product name. You can use multiple keywords by seperating each keyword with a space.

KEYWORDS = "nike jordan dunk"

Proxy:
This is an optional parameter where you can set your own proxies. There are two formats:
a) PROXY = "<proxy>:<port>"
b) PROXY = "<proxy_username>:<proxy_password>@<proxy_domain>:<port>"
You can use multiple proxies by seperating each proxy with a space.

PROXY = "000.000.000.00:111 222.222.222.22:333"

Free Proxy:
This is a boolean variable whether requests are made with free proxies.

FREE_PROXY = True

Details:
These are extra parameters that may be needed for specific monitors

LOCATION = "GB"

Some monitors require a few extra details. This section will mention those parameters and hwo they should be set.

Nike SNKRS
The SNKRS monitor can be used in 42 countries. To set it up for your location, please find the 2 letter country code and the language code in the table below.

Example: If I want the Belgium site, I find the corresponding country code and language code in the table. We see that the country code is "BE>", and there are 4 choices for the language (en-GB, nl, de or fr). Choose the language you want. In this case I want English, so will use en-GB. Below is how it should be set up 

LOC = "BE"
LAN = "en-GB"

Shopify
The Shopify monitor requires you to put in the URL for a specific shopify store. Note that the URL must point to a .json endpoint.

Example: For the Hanon Shop, if we want to monitor new items, the address is...
https://www.hanon-shop/collections/whats-new

To get all the products in a json format, we can use the following address...
https://www.hanon-shop/collections/whats-new/products.json
